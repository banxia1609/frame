## 项目简述

基于 VueCli3.0 + Vue2.0 + elementui 搭建的前端框架

## 环境和依赖

- node
- yarn
- webpack
- @vue/cli ~3

> 请注意，建议使用 [Yarn](https://yarnpkg.com/) 包管理工具


## 项目运行和发布

- 安装依赖

```
yarn install
```

- 开发模式运行

```
yarn start
```

- 编译项目

```
yarn build
```

- 发布项目

```
将编译完成的所有文件，即可部署文件
```

## 浏览器兼容

Modern browsers and IE10.
